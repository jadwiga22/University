Obliczanie wyrazów ciągu przy pomocy różnych metod¶

Mamy dany ciąg $ x_k = A(1+\sqrt{3})^k + B(1-\sqrt{3})^k $. Spełnia on zależność rekurencyjną $ x_k = 2(x_{k-1} + x_{k-2}) $. Przyjmujemy $ x_1 = 1, x_2 = 1 - \sqrt{3}$. Wówczas $ x_k = (1-\sqrt{3})^k$.

Definiujemy sposoby obliczania ciągu - odpowiednio FormulaA (wzór rekurencyjny) i FormulaB (wzór jawny), a także zmienną iter, czyli liczbę wyrazów ciągu, które chcemy obliczyć.

In [2]:
iter = 100

function FormulaA(X, i, type)
    X[i] = type(2) * (X[i-1] + X[i-2])
end

function FormulaB(X, i, type)
    X[i] = B*x2^i
end

function FormulaC(X, i, type)
    X[i] = eps(type)*C^i + B*x2^i
end

Obliczamy iter-wiele wyrazów ciągu przy użyciu przekazanego sposobu.

In [3]:
function CalculateSequence(type, formula)
    X = zeros(type, (iter,))
    X[1] = x1
    X[2] = x2
    
    for i in 3:iter
        X[i] = formula(X, i, type)
    end
    return X
end

Definiujemy kolejno funkcje RecursiveA, ExplicitB, ExplicitC, które obliczają ciągi zadane wzorami odpowiednio w podpunktach a, b i c.

In [4]:
function RecursiveA(type)
    CalculateSequence(type, FormulaA)
end

function ExplicitB(type)
    CalculateSequence(type, FormulaB)
end

function ExplicitC(type)
    CalculateSequence(type, FormulaC)
end

Na końcu w funkcji SolveWithPrecision definiujemy potrzebne stałe. Przyjmuje ona jako argument typ, w którym ma obliczać wynik.

In [5]:
function SolveWithPrecision(type)
    global x1 = type(1)
    global x2 = type(1 - sqrt(3))
    global A = type(0)
    global B = type(1.0 / (1 - sqrt(3)))
    global C = type(1 + sqrt(3))


    return [RecursiveA(type), ExplicitB(type), ExplicitC(type)]
end
